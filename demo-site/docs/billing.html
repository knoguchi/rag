Billing & Usage

Demo Cloud uses a pay-as-you-go billing model. You only pay for the resources you use, with no upfront costs or long-term commitments.

Billing Cycle

  • Bills are generated on the 1st of each month
  • Payment is due within 15 days
  • Usage is calculated hourly and aggregated monthly
  • All prices are in USD

Understanding Your Bill

Your monthly invoice includes charges for:

  • Compute: Running instances (per hour)
  • Storage: Object storage (per GB/month)
  • Database: Managed databases (per hour)
  • Bandwidth: Outbound data transfer (per GB)
  • Add-ons: Support plans, monitoring, etc.

Payment Methods

We accept:

  • Credit cards (Visa, Mastercard, American Express)
  • Debit cards
  • ACH bank transfer (US only)
  • Wire transfer (Enterprise plans)

Managing Payment Methods

# Add a credit card
demo billing payment-method add

# Set default payment method
demo billing payment-method default pm_abc123

# Remove a payment method
demo billing payment-method remove pm_old456

Cost Optimization

Reserved Instances

Save up to 50% by committing to 1 or 3 year terms:

Term Payment Discount
1 year Monthly 25%
1 year Upfront 35%
3 years Monthly 40%
3 years Upfront 50% 
# Purchase a reserved instance
demo billing reservation create \
  --type standard-4 \
  --term 1-year \
  --payment monthly \
  --quantity 2

Spot Instances

Save up to 80% on compute by using spare capacity:

demo compute create batch-worker \
  --type standard-4 \
  --spot \
  --max-price 0.05

Note: Spot instances can be terminated with 2 minutes notice when capacity is needed.

Budgets and Alerts

Set up budgets to track and control spending:

Create a Budget

demo billing budget create \
  --name "Monthly Limit" \
  --amount 500 \
  --period monthly \
  --alert-at 50,80,100

Budget Alerts

Receive notifications when spending reaches thresholds:

  • Email notifications to billing contacts
  • Slack integration
  • Webhook for custom integrations
# Configure alert channels
demo billing alerts configure \
  --email billing@company.com \
  --slack-webhook https://hooks.slack.com/...

Credits

Credits can be applied from:

  • Promotional offers
  • Referral program
  • Support credits
  • Partner programs
# View credit balance
demo billing credits

# Apply a promotional code
demo billing credits apply PROMO2024

Cost Allocation Tags

Tag resources to track costs by project, team, or environment:

# Tag resources at creation
demo compute create web-server \
  --tags "project=website,team=engineering,env=production"

# View costs by tag
demo billing usage --group-by tag:project

Enterprise Billing

Enterprise customers get additional features:

  • Consolidated billing: Single invoice for multiple accounts
  • Custom payment terms: Net 30/60/90 options
  • Dedicated billing contact: Your own account manager
  • Custom contracts: Volume discounts and SLAs
  • Purchase orders: PO-based billing support

Tax Information

Update your tax information for accurate invoicing:

demo billing tax-info update \
  --tax-id "US12-3456789" \
  --business-name "Demo Corp" \
  --address "123 Main St, San Francisco, CA 94105"

Tax exemption certificates can be uploaded in the console under Settings → Billing → Tax Exemption.
