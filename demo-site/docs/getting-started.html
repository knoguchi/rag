Getting Started with Demo Cloud

Welcome to Demo Cloud! This guide will help you set up your account and deploy your first application in minutes.

Prerequisites

  • An Demo Cloud account (sign up at cloud.demo.com)
  • The Demo CLI tool installed on your machine
  • Basic familiarity with command line interfaces

Step 1: Install the Demo CLI

The Demo CLI is the fastest way to interact with Demo Cloud services. Install it using your package manager:

macOS (Homebrew)

brew install demo-cloud/tap/demo-cli

Linux (apt)

curl -fsSL https://packages.demo.cloud/gpg | sudo gpg --dearmor -o /usr/share/keyrings/demo.gpg
echo "deb [signed-by=/usr/share/keyrings/demo.gpg] https://packages.demo.cloud/apt stable main" | sudo tee /etc/apt/sources.list.d/demo.list
sudo apt update && sudo apt install demo-cli

Windows (Chocolatey)

choco install demo-cli

Step 2: Authenticate

Log in to your Demo Cloud account using the CLI:

demo login

This will open a browser window for authentication. After logging in, your credentials will be stored securely on your machine.

Step 3: Create Your First Project

Create a new project to organize your resources:

demo project create my-first-project
demo project use my-first-project

Step 4: Deploy a Sample Application

Let's deploy a simple web application:

# Clone our sample app
git clone https://github.com/demo-cloud/sample-app.git
cd sample-app

# Deploy to Demo Cloud
demo deploy

The CLI will automatically detect your application type and configure the appropriate runtime.
